# Интернет-магазин на Django.

## Описание:

Интернет-магазин, разработанный на фреймворке Django (Python), предназначен для продажи товаров онлайн

## Установка и настройка:

### Клонируйте репозиторий:
```
https://github.com/hristyaa/DjangoWebShop.git
```
### Создайте виртуальное окружение и установите необходимые зависимости из `pyproject.toml`:
Выполните команду:
```
poetry install
```
Установка конкретных зависимостей (при необходимости):
```
poetry add django 
poetry add dotenv 
```

### Настройка базы данных
Скопируйте файл `.env.example` и переименуйте его в `.env`
Откройте `.env` и заполните ваши данные.

## Запуск сервера:
```
python manage.py runserver
```
После запуска сервера приложение будет доступно по адресу: `http://127.0.0.1:8000/`

### Доступные страницы:

- #### Главная
URL: `http://127.0.0.1:8000/home/`

Описание: Основная страница магазина Skystore 

Функционал: 
- Навигация по сайту; 
- Переход на страницу `Контакты`.

- #### Контакты
URL: `http://127.0.0.1:8000/contacts/`

Функционал:
- Основные контакты магазина Skystore; 
- Форма для отправки обратной связи, содержащая имя, телефон и сообщение клиента; 
- После осуществления обратной связи отображается сообщение об их успешной отправке данных;
- Переход на страницу `Главная`.

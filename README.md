# Интернет-магазин на Django.

## Описание:

Интернет-магазин, разработанный на фреймворке Django (Python), предназначен для продажи товаров онлайн

## Установка и настройка:

### Клонируйте репозиторий:
```
https://github.com/hristyaa/DjangoWebShop.git
```
### Создайте виртуальное окружение и установите необходимые зависимости из `pyproject.toml`:
Выполните команду:
```
poetry install
```
Установка конкретных зависимостей (при необходимости):
```
poetry add django 
poetry add dotenv 
poetry add psycopg2-binary
poetry add pillow
poetry add ipython
```

### Настройка базы данных
Скопируйте файл `.env.example` и переименуйте его в `.env`
Откройте `.env` и заполните ваши данные.

## Запуск сервера:
```
python manage.py runserver
```
После запуска сервера приложение будет доступно по адресу: `http://127.0.0.1:8000/`

### Доступные страницы:

- #### Главная (Каталог)
URL: `http://127.0.0.1:8000/

Описание: Основная страница интернет-магазина ТехноМания

Функционал: 
- Навигация по сайту; 
- Каталог товаров (С главной страницы можно перейти на страницу отдельного товара и обратно, например URL:`http://127.0.0.1:8000/products/1/`) ;
- Переход на страницу `Контакты`;
- Переход на страницу `Блог`.

- #### Контакты
URL: `http://127.0.0.1:8000/contacts/`

Функционал:
- Основные контакты интернет-магазина ТехноМания; 
- Форма для отправки обратной связи, содержащая имя, телефон и сообщение клиента; 
- После осуществления обратной связи отображается сообщение об их успешной отправке данных;
- Переход на страницу `Главная`;
- Переход на страницу `Блог`.

- #### Блог
URL: `http://127.0.0.1:8000/blogs/posts/`

Функционал:
- На странице выводится список постов, которые имеют положительный признак публикации;
- При открытии отдельной статьи увеличивается счетчик просмотров статьи;
- Когда пост достигает 100 просмотров, отправляется на почту поздравление с достижением (заполнить данные в `.env.example`) .
- Предусмотрена возможность добавления поста, редактирования и удаления имеющихся постов;
- После успешного редактирования записи происходит перенаправление пользователя на страницу отредактированного поста.
